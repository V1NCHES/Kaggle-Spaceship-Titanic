# Spaceship Titanic: Оптимизированное решение (Ensemble Learning)

Этот репозиторий содержит продвинутое решение задачи **Spaceship Titanic** с использованием современных алгоритмов градиентного бустинга и оптимизированного пайплайна оценки моделей.

## 🚀 Ключевые особенности решения

1.  **Продвинутый Feature Engineering**:
    *   **Family Imputation**: Восстановление `HomePlanet` на основе связей по фамилиям и группам.
    *   **Spatial Analysis**: Зонирование кабин по регионам для учета физического расположения пассажиров.
    *   **Log Transformation**: Нормализация финансовых признаков (траты на борту).

2.  **Эффективный Ансамбль (Voting Classifier)**:
    *   Сравнение 10 конфигураций: 6 индивидуальных моделей (**CatBoost, XGBoost, LightGBM, Random Forest, Logistic Regression, Decision Tree**) и 4 различных ансамблей.
    *   Автоматический выбор лучшей модели по метрике Accuracy.

3.  **Оптимизация производительности и памяти**:
    *   **Memory-Safe Pipeline**: Использование облегченных параметров на этапе кросс-валидации и автоматическое переключение на полные параметры при финальном обучении.
    *   **Garbage Collection**: Принудительная очистка RAM после каждой итерации для предотвращения MemoryError.

4.  **Комплексная оценка (5 Метрик)**:
    *   Модели оцениваются не только по **Accuracy**, но и по **Precision, Recall, F1-score** и **ROC-AUC** для детального анализа качества.
